To execute the application, you should open the file "execute.bat" (if not using Windows, see NOTE 1). After this, the first administrator is automatically created with the email "admin@t4j.com" and the password is sent to the file "e-mail.txt".

NOTE 1: if not using Windows, run the application on an IDE or in command line with the command "java -jar payments-app.jar" in the folder that contains this JAR file.
NOTE 2: if the application is not running properly, run on an IDE
